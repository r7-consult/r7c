# Добро пожаловать в R7 Plugin Manager

R7 Plugin Manager — менеджер установки и обновления плагинов для экосистемы R7.

## Что нового

- Полностью обновлённый интерфейс в стиле VS Code.
- Улучшенный UX карточек, поиска и предпросмотра.
- Динамическая загрузка Welcome/License контента из GitHub.
- Поддержка светлой и тёмной темы (по умолчанию — тёмная).

## Возможности

- Установка и обновление плагинов в одном окне.
- Просмотр changelog и описаний плагинов.
- Быстрый поиск по каталогу.
- Встроенная вкладка с лицензией менеджера.
