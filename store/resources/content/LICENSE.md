# Лицензия менеджера плагинов R7 Consult

Данный файл используется во вкладке **License** внутри Plugin Manager.

## Временная версия

Текст лицензии пока не финальный. Пожалуйста, замените содержимое этого файла на утверждённую юридическую версию.

## Использование

- Кнопка **«Лицензия»** в интерфейсе открывает popup на вкладке **License**.
- Контент подгружается сначала из GitHub (ветка `main`, затем `master`),
  затем из локального `store/resources/content/LICENSE.md`.
- Если загрузка недоступна, показывается встроенный fallback.
